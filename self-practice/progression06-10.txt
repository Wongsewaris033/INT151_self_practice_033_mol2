1.CRUD
ตัวแอปฯ มีการสร้างชุดฟังก์ชันหลักเพื่อจัดการข้อมูลคำคมที่อยู่ในรูปของarray:
-addQuote: สำหรับเพิ่มคำคมใหม่ 
-deleteQuote: สำหรับลบคำคมออกจากรายการ โดยใช้ findIndex และ splice (หรือจะใช้ filter ก็ได้) 
-updateQuote: สำหรับแก้ไขเนื้อหาหรือผู้เขียนของคำคมเดิม โดยใช้เทคนิคการผสานข้อมูล (Spread Operator) 
-getAllQuotes: สำหรับดึงข้อมูลคำคมทั้งหมดมาแสดง 

2.DOM Manipulation
มีการเรียนรู้วิธีทำให้ข้อมูลแสดงบนหน้าจอและรับการโต้ตอบจากuser:
สร้าง UI: สร้าง div และ p เพื่อแสดงคำคมแต่ละอัน และสร้างปุ่ม Edit กับ Delete ที่มี ID ของคำคมกำกับไว้ 
interact: ใช้ Event Delegation คือการฟังเหตุการณ์ไว้ที่ตัวหลัก (quoteList) เพื่อจัดการกับการคลิกปุ่ม Edit (ดึงข้อมูลมาใส่ฟอร์ม) และ Delete (ลบข้อมูลทั้งจากโค้ดและหน้าจอ) 
จัดการฟอร์ม: ควบคุมการส่งข้อมูลจากฟอร์มเพื่อเพิ่มหรือแก้ไขคำคม โดยไม่ให้หน้าเว็บรีโหลด 

3.Element
มีการศึกษาความแตกต่างของวิธีการใส่เนื้อหาใน Element:
innerHTML: จะเข้าใจและตีความโค้ด HTML ที่ใส่เข้าไป (เช่น ใส่ <i> จะเห็นเป็นตัวเอียง) 
innerText: จะไม่เข้าใจโค้ด HTML แต่จะแสดงข้อความตามที่ผู้ใช้เห็น โดยยังเข้าใจ CSS 
textContent: จะไม่เข้าใจอะไรเลย มองทุกอย่างเป็นตัวอักษรธรรมดา 
ค้นหา: แสดงวิธีใช้ querySelector, querySelectorAll และ getElementById รวมถึงการจัดการ Class (add, remove, toggle) และการดึงค่าจาก data attribute 

4.Local Storage
มีการเพิ่มความสามารถให้เว็บ "จำ" สิ่งต่างๆ ไว้ได้ แม้ผู้ใช้จะปิดหรือรีเฟรชหน้าไปแล้ว:
จำสไตล์ที่ตั้งค่า: บันทึกค่าสีพื้นหลัง, สีตัวอักษร, และขนาดตัวอักษรที่ผู้ใช้เลือกไว้ใน localStorage เมื่อเปิดเว็บใหม่ก็จะใช้สไตล์นั้นทันที 
จำรายการคำคม: ใช้ JSON.stringify และ JSON.parse เพื่อแปลงรายการคำคมให้เป็นข้อความแล้วบันทึกไว้ใน localStorage และดึงกลับมาใช้เมื่อโหลดหน้าเว็บ เพื่อให้คำคมที่เพิ่มไปไม่หาย 
การรีเซ็ต: เมื่อกดปุ่มรีเซ็ต จะลบข้อมูลสไตล์ออกจาก localStorage และตั้งค่าทุกอย่างกลับเป็นค่าเริ่มต้น
